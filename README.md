# BIRTHDAY_PRESENTS terminal commands

On Mac: 

(1) clang++ -arch arm64 -std=c++11 -o birthday_presents birthday_presents.cpp

(2) ./birthday_presents

On Windows:

(1) gcc -std=c++11 -o birthday_presents birthday_presents.cpp

(2) ./birthday_presents
